Google Scatter Chart

Creating scatter plots (or point charts) is easy. The following example generates 1000 points by multiplying two numbers for the X coordinate and two numbers for the Y coordinate:

1: 
2: 
3: 
4: 
5: 

let rnd = new System.Random() 
let next() = rnd.NextDouble() * rnd.NextDouble()
let points = [ for i in 0 .. 100 -> next(), next() ]

points |> Chart.Scatter
